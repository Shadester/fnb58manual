## What is the FNB58?

The FNIRSI FNB58 is a high-precision USB voltage and current detection meter with fast charging protocol support. It features:

- 2.0-inch color TFT display with auto-rotation
- Support for multiple fast charging protocols (PD, QC, FCP, SCP, AFC, VOOC, etc.)
- Cable resistance measurement
- E-Marker chip reading
- Real-time PD protocol monitoring
- Power measurement up to 120W (28V, 7A)
